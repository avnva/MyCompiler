Тема работы

Разработка текстового редактора с функциями языкового процессора. Объявление и инициализация лямбда-выражений в языке программирования Python.

Особенности языка

Лямбда-выражения в языке Python - это анонимные функции, которые могут содержать только одно выражение. Они обычно используются вместе с функциями высшего порядка, такими как map(), filter() и reduce(). Формат записи лямбда-выражений в Python следующий:

имя_переменной = lambda аргументы: выражение

Примеры верных строк из языка

В соответствии с разработанной грамматикой синтаксический анализатор (парсер) лямбда-выражений в Python будет считать верными следующие записи:

sum = lambda a, b: a + b
multiply = lambda x, y: x * y
minus = lambda x, y: x - y
div = lambda x: x / 2
