Лямбда-выражение в языке Python

Лямбда-выражение в языке Python — это анонимная функция, определенная с использованием ключевого слова "lambda". Она позволяет создавать короткие функции без необходимости объявления их через ключевое слово "def".

Формат записи для объявления лямбда-выражения в Python:

имя_функции = lambda [параметры] : инструкция;

Примеры верных строк из языка:

sum = lambda a : a + 1;
min = lambda a, b : a - b;
m = lambda a, b: (a * b) + 10;
div = lambda a, b, c: (a + b - c) / 10.66;
x = lambda a : -a * 10000.8;
