Грамматика

Определим грамматику лямбда-выражения формата «sum = lambda a,b:a+b;» языка Python G[<ЛВ>] в нотации Хомского с продукциями P:

G[<ЛВ> = <лямбда выражение>]:

        P = {
            1. <ЛВ> → Б<ИД>
            2. <ИД> → Б | Ц <ИД> | '=' <Л>
            3. <Л> → 'lambda' <ПРОБЕЛ>
            4. <ПРОБЕЛ> → ' ' <АРГУМ> | ':' <ФУНКЦ>
            5. <АРГУМ> → Б<ИМЯАРГУМЕНТА>
            6. <ИМЯАРГУМЕНТА> → Б | Ц <ИМЯАРГУМЕНТА> | ',' <АРГУМ> | ':' <ФУНКЦ>
            7. <ФУНКЦ> → Б<ИМЯОПЕРАТОРА1>
            8. <ИМЯОПЕРАТОРА1> → Б | Ц<ИМЯОПЕРАТОРА1> | (+|-|*|/) <ИМЯОПЕРАТОРА2>
            9. <ИМЯОПЕРАТОРА2> → Б<ИМЯОПЕРАТОРА2ОСТ>
            10. <ИМЯОПЕРАТОРА2ОСТ> → Б | Ц<ИМЯОПЕРАТОРА2ОСТ> | ';'
            11. <Б> → 'a' | 'b' | ... | 'z' | 'A' | 'B' | ... | 'Z'
            12. <Ц> → ‘0’ | ‘1’ | … | ‘9’

Следуя введенному формальному определению грамматики, представим G[<ЛВ>] ее составляющими:

        <ЛВ> = <лямбда-выражение>;
        VT = {'lambda', ' ', '=', ',', ':', '+', '-', '*', '/', ';'}
        VN = {'<ЛВ>', '<ИД>', '<Л>', '<ПРОБЕЛ>', '<АРГУМ>', '<ИМЯАРГУМЕНТА>', '<ФУНКЦ>', '<ИМЯОПЕРАТОРА1>', '<ИМЯОПЕРАТОРА2>', '<ИМЯОПЕРАТОРА2ОСТ>'}

Для обработки арифметических выражений мной была сформулирована следующая грамматика, которая уже не является автоматной:

        1. <Expression> → <Additive> ';'
        2. <Additive> → <Multiplicative> {('+' | '-') <Multiplicative>}
        3. <Multiplicative> → <Primary> {('*' | '/') <Primary>}
        4. <Primary> → '(' <Expression> ')' | <Number> | <Identifier> | '-' <Primary>
        5. <Number> → <Integer> | <Float>
        6. <Integer> → <Digit> {<Digit>}
        7. <Float> → <Integer> '.' <Integer>
        8. <Identifier> → <Letter> {<LetterOrDigit>}
        9. <Letter> → 'a' | 'b' | ... | 'z' | 'A' | 'B' | ... | 'Z'
        10. <LetterOrDigit> → <Letter> | <Digit>
        11. <Digit> → '0' | '1' | ... | '9'
