Грамматика Лямбда-выражения Python

Определим грамматику лямбда-выражения формата «имя_функции = lambda [параметры] : инструкция;» языка Python G[<LF>] в нотации Хомского с продукциями P:

        G[<LF> = <lambda function>]:
        P = {
        1. <LF> -> letter<ID><ASSIGN>
        2. <ASSIGN> -> '='<LAMBDA>
        3. <LAMBDA> -> 'lambda'<ARG>
        4. <ARG> -> letter<ID><FUNC>|letter<ID><ENUM>
        5. <ENUM> -> ','<ARG>
        6. <FUNC> -> ':'<EXPRESSION>
        7. <EXPRESSION> -> <ADDITIVE>';'
        8. <ADDITIVE> -> <MULTIPLICATIVE><ADD>
        9. <ADD> -> '+'<MULTIPLICATIVE><ADD>|'-'<MULTIPLICATIVE><ADD>
        10. <MULTIPLICATIVE> -> <PRIMARY><M>
        11. <M> -> '*<PRIMARY><M>|'/'<PRIMARY><M>
        12. <PRIMARY> -> '('<EXPRESSION>')'|<NUMBER>|letter<ID>|'-'<PRIMARY>
        13. <NUMBER> -> <INT>|<FLOAT>
        14. <INT> -> digit<INT>
        15. <FLOAT> -> <INT>'.'<INT>
        16. <ID> -> letter<ID>|digit<ID>|'_'<ID>|ε;
        17. letter -> 'a'|'b'|...|'z'|'A'|'B'|...|'Z'
        18. digit -> '1'|...|'9'
        }

Следуя введенному формальному определению грамматики, представим G[<LF>] ее составляющими:

        <LF> = <lambda function>;
        VT = {’lambda’, ’=’, ’,’, ’:’, ’+’, ’-’, ’*’, ’/’, ’;’, ‘a’…’z’, ‘A’…’Z’, ‘0’…’9’}
        VN = {<LF>, <ID>, <ASSIGN>, <LAMBDA>, <ARG>, <ENUM>, < FUNC>, 
        <EXPRESSION>, <ADDITIVE>, <ADD>, <MULTIPLICATIVE>, <M>, <PRIMARY>, 
        <NUMBER>, <INT>, <FLOAT>}
